# Active Directory Account Audit and Support Toolkit

## Overview
This PowerShell project was created during MSSA Project Week to automate common Active Directory account support tasks in a lab environment.

## Features
- Show locked accounts
- Unlock a specific user account
- Show expired accounts
- Show inactive accounts older than 30 days

## Tools Used
- PowerShell 7
- Visual Studio Code
- Git
- GitHub
- Active Directory lab environment

## Purpose
The purpose of this project is to demonstrate basic PowerShell scripting skills while solving practical administrative tasks in Active Directory.

## How to Run
1. Open PowerShell on a system with Active Directory tools installed
2. Navigate to the script folder
3. Run the script:
   ```powershell
   .\AD-Account-Toolkit.ps1
